# Vibe Coding vs AI Pair Programming

## 📌 Project Overview
This project compares two approaches to building a Task Manager app:
- Vibe Coding (using v0)
- AI Pair Programming (using Vite + React)

---

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Filter tasks (All, Active, Completed)

---

## 🛠️ Tech Stack
### Vibe Version
- v0 tool
- Next.js

### Pair Version
- Vite
- React
- JavaScript

---

## 🌐 Live Deployment
- Pair version: https://sparkly-tulumba-5c1db2.netlify.app

---

## 🔍 Key Learnings
- Vibe coding is fast and useful for quick UI generation
- Pair programming gives better understanding and control
- Debugging is easier in pair programming approach

---

## 📂 Project Structure
- vibe-version → generated using v0
- pair-version → manually built using React

---

## 🎯 Conclusion
Vibe coding is best for speed and prototyping.  
AI pair programming is better for learning, debugging, and real-world development.
